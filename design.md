# VideoEdit Pro - デザイン設計書

## アプリ概要
iOS/Android向けの動画編集アプリ。デバイスから動画を選択し、トリミング・フィルター適用・テキストオーバーレイ・BGM追加などの編集を行い、完成した動画をエクスポートできる。

## 画面一覧

### 1. ホーム画面 (Home)
- **タブ**: 「ホーム」アイコン
- **コンテンツ**: 
  - ヘッダー: アプリロゴ + タイトル
  - 「新しいプロジェクト」ボタン（大きなCTAカード）
  - 最近のプロジェクト一覧（サムネイル付きリスト）
  - 各プロジェクトカード: サムネイル、タイトル、日時、長さ

### 2. プロジェクト一覧画面 (Projects)
- **タブ**: 「フォルダ」アイコン
- **コンテンツ**:
  - グリッドレイアウトでプロジェクト一覧表示
  - 各プロジェクト: サムネイル、タイトル、作成日
  - スワイプで削除
  - 空状態: イラスト + 「最初のプロジェクトを作成」メッセージ

### 3. 動画選択画面 (Video Picker)
- **アクセス**: ホーム画面の「新しいプロジェクト」から遷移
- **コンテンツ**:
  - デバイスのメディアライブラリから動画を選択
  - expo-image-pickerを使用
  - 選択後、エディター画面へ遷移

### 4. エディター画面 (Editor) - メイン編集画面
- **レイアウト** (上から下):
  - **プレビューエリア**: 動画プレビュー（画面上部60%）
    - 再生/一時停止ボタン
    - 現在時間表示
  - **タイムライン**: 動画のサムネイルストリップ（スクロール可能）
    - 再生ヘッド（現在位置インジケーター）
    - トリミングハンドル（左右）
  - **ツールバー**: 編集ツールアイコン一覧（画面下部）
    - トリミング、フィルター、テキスト、音楽、速度

### 5. フィルター選択パネル (Filters)
- **アクセス**: エディターのツールバーから
- **コンテンツ**:
  - 横スクロールのフィルタープレビュー一覧
  - フィルター名: オリジナル、ビビッド、モノクロ、セピア、ウォーム、クール、ドラマチック、フェード
  - 選択中のフィルターにハイライト表示
  - 強度スライダー

### 6. テキスト追加パネル (Text Overlay)
- **アクセス**: エディターのツールバーから
- **コンテンツ**:
  - テキスト入力フィールド
  - フォントサイズスライダー
  - テキストカラーピッカー（プリセットカラー一覧）
  - テキスト位置: 上部/中央/下部
  - テキストスタイル: 太字、イタリック

### 7. BGM追加パネル (Music)
- **アクセス**: エディターのツールバーから
- **コンテンツ**:
  - プリセットBGMリスト（カテゴリ別）
  - 各トラック: タイトル、長さ、再生プレビューボタン
  - 音量スライダー
  - カテゴリ: ポップ、チル、エピック、アコースティック

### 8. 速度調整パネル (Speed)
- **アクセス**: エディターのツールバーから
- **コンテンツ**:
  - 速度スライダー (0.25x ~ 3.0x)
  - プリセットボタン: 0.5x, 1.0x, 1.5x, 2.0x

### 9. エクスポート画面 (Export)
- **アクセス**: エディターの「エクスポート」ボタンから
- **コンテンツ**:
  - プログレスバー
  - エクスポート設定（品質選択）
  - 完了後: 共有ボタン、ギャラリーに保存ボタン

## 主要ユーザーフロー

### フロー1: 新規プロジェクト作成
1. ホーム画面で「新しいプロジェクト」をタップ
2. デバイスの動画ライブラリが開く
3. 動画を選択
4. エディター画面に遷移
5. 編集ツールを使用して動画を編集
6. 「エクスポート」をタップ
7. エクスポート設定を確認
8. エクスポート完了 → 共有 or ギャラリー保存

### フロー2: 動画トリミング
1. エディター画面でタイムラインのハンドルをドラッグ
2. 開始/終了位置を調整
3. プレビューで確認
4. 「適用」をタップ

### フロー3: フィルター適用
1. ツールバーの「フィルター」をタップ
2. フィルター一覧が表示
3. フィルターをタップしてプレビュー
4. 強度を調整
5. 「適用」をタップ

### フロー4: テキスト追加
1. ツールバーの「テキスト」をタップ
2. テキスト入力パネルが表示
3. テキストを入力
4. スタイル（色、サイズ、位置）を調整
5. 「適用」をタップ

### フロー5: BGM追加
1. ツールバーの「音楽」をタップ
2. BGMカテゴリを選択
3. トラックをプレビュー再生
4. トラックを選択
5. 音量を調整
6. 「適用」をタップ

## 技術的な考慮事項

- **動画再生**: expo-video を使用
- **動画選択**: expo-image-picker を使用
- **サムネイル生成**: expo-video-thumbnails を使用
- **音声再生**: expo-audio を使用（BGMプレビュー）
- **ハプティクス**: expo-haptics を使用
- **状態管理**: React Context + useReducer でプロジェクト状態を管理
- **ローカル保存**: AsyncStorage でプロジェクトメタデータを保存
- **アニメーション**: react-native-reanimated でスムーズなUI遷移
- **ジェスチャー**: react-native-gesture-handler でタイムライン操作

## レイアウト原則

- ダークテーマベース（動画編集アプリの標準）
- 片手操作を考慮したツールバー配置（画面下部）
- プレビューエリアを最大化
- ツールパネルはボトムシートで表示（画面遷移を最小化）
- タイムラインは水平スクロールで直感的操作
